## Tech Stack
- React Native (Expo)
- TypeScript
- AsyncStorage
- Expo Speech

## Features

- Choose your accent (Classic, British, Italian)
- Adjustable humor levels (Low, Medium, High)
- Auto callouts for nearby places (coffee, restaurants, gas)
- Settings persist after refresh
- Smooth iOS-style UI

- Voice with unlimited high-quality TTS should be coming from our own staff. Relying on third-party presets will cost a lot per usage.
